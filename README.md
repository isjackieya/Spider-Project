## auto_login.py

> 通过对校园网登录的接口发送请求包，实现自动登录
>
> 使用自启动，可以实现校园网自动登录
>
> 新建一个bat程序
>
> 写入以下内容


```
cd  [py程序所在目录]
python auto_login.py
```

>  将该bat文件放在计划任务中即可实现自启动
